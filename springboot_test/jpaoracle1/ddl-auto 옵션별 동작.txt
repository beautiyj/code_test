create 			: 서버 켤 때마다 테이블을 삭제 후 재생성 (기존 데이터 다 날아감)
create-drop		: 서버 켤 때 생성, 종료할 때 삭제
update			: 테이블 없으면 생성, 있으면 변경분만 반영 (컬럼 추가 등) — 가장 많이 씀
none			: 아무것도 안 함 (운영 환경에서 주로 사용)

## JPA Setting  # create create-drop update none ddl-auto 옵션별 동작
#spring.jpa.hibernate.ddl-auto=update 
#spring.jpa.generate-ddl=false
#spring.jpa.show-sql=true
#spring.jpa.database=oracle
#spring.jpa.database-platform=org.hibernate.dialect.OracleDialect
