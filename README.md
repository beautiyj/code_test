# 🚀 code_test
> **Intellij를 활용하여 Python, Java, Kotlin 형태로 문제 풀기**

---

### 📂 폴더 구조
* **task 폴더**: 기출 문제 위주
* **test 폴더**: 기초 및 입문용 연습 문제 위주
* **공통**: 같은 예제를 각기 다른 언어로 풀이 진행

---

### 📚 사용 교재
* **Python**: [혼자 공부하는 파이썬]
* **Java**: [혼자 공부하는 자바]
* **Kotlin**: [코틀린 완벽 가이드] 또는 [안드로이드 앱 개발서]
  * *팁: 코틀린은 아예 모른다면 전자를, 지식이 있다면 후자를 활용할 것! 후자는 안드로이드 개발서에 가깝다.*
